Tester Performance Tracker

This project was done in 2024. This is a temporary soluction to track the testers performances by backward timeframe from now, and able to generate the data in Excel file.

Functionalities are:

  • Getting a data from csv file stored in Tracking folder.
  • A text box where the user can enter the number of hours of they want to see the data from now on
  • A yeild data by testers within the timeframe that the user enter will be display with three light indicators that represent performances of the testers or media defects
  • A pack table shows the pack number together with associated hardware and failure bins
  • The pack number in a pack table opens the met files when the user clicks on it.
  • The RCC summary table displays the testers, datetime of tester failed and RCC message
  • The `Export data file` on app export a data file with Excel extension file.
